# Handoff Template

## Task
What was requested?

## Relevant Files
List only files directly related to the task.

## Current Behavior
What does the code do now?

## Root Cause / Goal
What is broken or what needs to be built?

## Constraints
- no unintended UI changes
- preserve architecture
- minimum change only
- keep routes stable

## Proposed Change
Describe the smallest safe implementation.

## Risks
List assumptions, possible regressions, or unknowns.

## Validation
What should be tested after implementation?
